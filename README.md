## Getting Started

### Installation

1. Clone the repository.
```sh
git clone https://github.com/nandinibansal2000/Plants-Vs-Zombie-Game.git
```
2. Unzip the folder fxTry.zip

3. Run the following command to run the game 
```sh
javac App.java
```
